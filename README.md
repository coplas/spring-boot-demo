# spring-boot-demo

Simple demo to test spring boot

* querydsl
* bootstrap
* jpa (h2)
* thymeleaf
* thymeleaf-spring-data-dialect (layouts and fragments)


This demo app generates test data at start, shows pagination, sorting and search (case insentive substrings).
